## Projects

### claude-pr-review-team
AI-powered code review tool that uses Claude to provide automated, high-quality pull request reviews for teams. Integrates with GitHub Actions for seamless CI/CD workflow. Supports multi-file analysis, architectural feedback, and actionable suggestions with code examples.

### podcastcut-skills
AI-driven podcast editing tool that automatically identifies and cuts segments of interest from podcast audio using speech recognition and language models. Supports multiple audio formats and handles long-form podcast episodes efficiently.

### deepagents-quickstarts
Quickstart templates and examples for building deep learning-based AI agents. Includes patterns for tool-using agents, multi-agent collaboration systems, and agents that learn from feedback using reinforcement learning.

### tpu_training
Framework and utilities for training large-scale deep learning models on Google TPUs. Includes distributed training strategies, mixed precision support, data pipeline optimization, and checkpoint management. Supports PyTorch/XLA.

### FMCW-DopplerPointTransformerNet
A Transformer-based neural network for processing FMCW radar Doppler point clouds. Designed for autonomous driving perception tasks such as object detection and velocity estimation. Leverages Doppler information to improve 3D perception in adverse weather conditions.

### dinov2-with-rope
Extension of Meta's DINOv2 self-supervised vision transformer with Rotary Position Embedding (RoPE). Provides relative positional encoding for generalization to varying image resolutions without retraining. Improves spatial reasoning for downstream tasks like object detection and segmentation.

### podcast-transcriber-mcp
A Model Context Protocol (MCP) server for transcribing podcasts. Provides podcast transcription as a tool that AI assistants can call to process and analyze podcast content. Built following the MCP specification for seamless integration with Claude and other MCP-compatible AI assistants.
